# md2pdf

## Markdown Support:

- Paragraphs
- Headers
  - Links
- Emphasis
  - Bold
  - Italic
  - Code
  - Strikethrough
- Links
- Lists
  - Unordered
  - Ordered
  - Nested
  - Definition lists
- Images
  - Links in images
- Code
  - Inline Code (backticks)
  - Fenced Code Blocks
- Tables
- Blockquotes
- Footnotes
- ~~Horizontal Rule~~
